🎮 Xbox Dashboard Lab - DIO
Este repositório contém o projeto prático desenvolvido durante o bootcamp da Digital Innovation One (DIO). O objetivo foi recriar a experiência visual e funcional da dashboard do Xbox, aplicando conceitos de design de interface e organização de dados.
🚀 Sobre o Projeto
O projeto simula a interface de um console de jogos, focando em:
Experiência do Usuário (UX): Navegação intuitiva e visual imersivo.
Design de Interface (UI): Uso de cores, ícones e tipografia fiéis à marca Xbox.
Exibição de Dados: Organização de informações de forma clara e dinâmica.
🛠️ Tecnologias Utilizadas
Excel/Power BI (ou HTML/CSS, dependendo da sua trilha) para a construção visual.
Tabelas Dinâmicas para filtragem de informações.
Segmentação de Dados (Slicers) para interatividade.
